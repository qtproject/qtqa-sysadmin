Qt Project Test Results

Test results for the Qt Project are published on this server.

The following results are currently available:

  • Coin - The Qt Continuous Integration System's (Coin's) dashboard
  • Logs - Build and autotests logs from Coin's builds
  • QtMetrics - autotest metrics from the Qt Continuous Integration System - currently under development
  • Old logs - Build and autotests logs from old Jenkins builds
  • Coverity scan results - External - Coverity scan results of Qt5
  • OpenCoverage - External - Open coverage results of Qt5

For more information, see the Qt Project wiki.
