Qt Project Test Results

Test results for the Qt Project are published on this server.

The following results are currently available:

  • ci - raw logs from the Qt Continuous Integration System used on codereview.qt-project.org
  • qt-metrics - autotest metrics from the Qt Continuous Integration System
  • release-testing - release testing form used to report status of packages from download.qt-project.org
  • status - status information about the Qt Continuous Integration System

For more information, see the Qt Project wiki.
